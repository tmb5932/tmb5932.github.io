My Projects

KwziiCasino

Personal Project

KwziiCasino is a fun and interactive project I designed to simulate various casino games. Built using JavaFX, this project aims to provide an engaging experience with multiple games and activities. It includes games such as blackjack, horse betting, slots, coin flipping, and even some miscellaneous like a solvable Rubik’s Cube.

Minecraft Mod

Personal Project

Currency Mod is a Minecraft mod that significantly enhances Minecraft’s trading mechanics and introduces a wealth of new content. The mod features custom villager trades, unique items, custom storage items, and new interactive blocks. With custom menus, block textures and crafting recipes, this mod is my exploration and experimentation with Minecraft modding.
